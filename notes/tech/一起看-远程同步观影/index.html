    一套可以和朋友一起远程看视频的开源解决方案。

    syncTV 远程同步观影

    • SyncTV 是一个允许您远程一起观看电影和直播的程序。git
    • 安装方式：脚本、docker、helm 等。
    • 默认端口：8080

    cpolar 内网穿透

    如果没有服务器，在本地搭建服务，一般只能在局域网内通信，外网无法访问。cpolar 可以轻松实现内网穿透。

    • 官网： https://www.cpolar.com/docs ，cpolar是一种安全的内网穿透云服务，它将内网下的本地服务器通过安全隧道暴露至公网。使得公网用户可以正常访问内网服务。
    • 步骤：
      • 在官网注册登陆，获取token。
      • 下载/安装。
      • 服务启动： cpolar service [start stop restart install uninstall]
      • 验证token: ./cpolar token {your token}
      • 开启隧道: ./cpolar http {your server port}, 这样就可以把你本地的服务发布到公网。
    • cpolar 服务的默认端口 9200，服务启动后可以通过 http://localhost:9200 访问。
      • 二级域名预留、自定义域名等，需要付费。免费服务包括1个进程4个隧道，随机域名。
